# anStats
Desktop monitor tool for android phone

## Impl
Nodejs + electron

## 功能：
*CPU频率
*CPU Top （total/user/system/avg）
*温度 （battery/CPU）
*背光亮度

## 限制：
*需userdebug版本

## 使用：
*直接解压，双击exe即可
*插上USB线后，点击第一个button连接手机即开始
*第二个button是用来切换充电功能的，可以disable/enable charging
*要切换手机，目前只能关闭重新开启后连接
